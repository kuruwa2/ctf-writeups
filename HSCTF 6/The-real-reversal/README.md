## Solution
Due to the hint, try to reverse `everything`. Command line tool `xxd` will do the job. Run

`< reversed.txt xxd -p -c1 | tac | xxd -p -r > WOW.txt`

and read into [WOW.txt](./WOW.txt) we could see the *reversed* flag.
